Los Atributos

son caracteristicas de las etiquetas o de los elementos html

pueden o no contener valores, si contienen valores debemos escrbir el nombre del atributo = valor o valores de ese atributo

no hay limite en la cantidad de atributos de una etiqueta o elemento, pero si hay algunos atributos que son obligatorios como :
♦ src: source en "img"
♦ href en "a"
♦ type en "input"
